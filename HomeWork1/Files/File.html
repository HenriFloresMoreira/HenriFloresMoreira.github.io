Creating the HTML File

To begin typing away we first need the HTML file. This is created within the project we made in the previous step.

Step 1: Within the project file, there is a folder called Web Content. Right click on it to see the options. Hover over the new tab and then select the HTML file.

Step 2: A menu will be pulled up and that is where you will name the file. Clicking the finish button creates the file.

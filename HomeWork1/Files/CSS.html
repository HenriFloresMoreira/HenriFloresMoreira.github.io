Creating the CSS File

Similar to creating the HTML file, the CSS file goes through a similar path to create it.

Step 1: Within the project file, there is a folder called Web Content. Right click on it to see the options. Hover over the new tab and then select the Other option.

Step 2: A menu will open up like this image.

Step 3: Scroll down the menu to find the Web folder. Click the arrow to lower the menu and click on the option called CSS File.

Step 2: A new menu will be pulled up and that is where you will name the file. Clicking the finish button creates the file.
